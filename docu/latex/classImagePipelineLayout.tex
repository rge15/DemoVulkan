\doxysubsection{Detailed Description}
Image Pipeline Layout which have a single descriptor set \& push constants. 

It contains base push constants for each shader as\+:


\begin{DoxyItemize}
\item Absolute\+Time \+: Total Time of execution of the program
\item Relative\+Time \+: Total Time of execution of the demo
\item Window\+Resolution \+: Resolution of the screen
\end{DoxyItemize}

The Descriptor set that contains manages 1 common image for reading/writing between all fotograms of the app
