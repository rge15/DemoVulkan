\doxysubsection{Detailed Description}
Basic Pipeline Layout which doesn\textquotesingle{}t have description sets. 

It contains base push constants for each shader as\+:


\begin{DoxyItemize}
\item Absolute\+Time \+: Total Time of execution of the program
\item Relative\+Time \+: Total Time of execution of the demo
\item Window\+Resolution \+: Resolution of the screen
\end{DoxyItemize}
