\doxysubsection{Detailed Description}
Base class for Render Pass Managers to inherit from. 

It inhers from Base Renser Resource class Manages the common creation parts of all the Render Passes

Modify the base creation info from child classes before creation

In this engine version there is just 1 REnder\+Pass\+Mng obj
